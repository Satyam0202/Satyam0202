# Personal Document Vault

This repository contains a Node.js + Express app for managing personal documents.

## Quick GitHub Pages demo
A simple static demo is available in the `docs/` folder. To publish it via GitHub Pages:

1. Push your repo to GitHub (already done).
2. In repository Settings → Pages, set source to `main`/`master` branch and `docs/` folder.
3. Save — GitHub will publish the static demo at `https://<your-username>.github.io/<repo-name>/`.

Note: The static `docs/` demo does not include server-side upload features. For the full app deploy, use Render or Railway.
