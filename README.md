# City-Sathi 🏙️

**Your all-in-one city companion — explore, navigate, and live smarter.**

A comprehensive web platform designed to help users discover and navigate cities with ease. Built specifically for Chandigarh, City-Sathi provides essential information about local services, popular places, hotels, hospitals, and more.

## 🌟 Features

### Core Functionality
- **Real-time Information**: Live weather updates, current time, and date display
- **Location Services**: Current location detection and distance calculations
- **Search & Discovery**: Comprehensive search for hotels, routes, restaurants, and services
- **Quick Actions**: Fast access to transportation options (Walk, Drive, Bike, Train, Uber, Ola)

### Services Covered
- 🏨 **Hotels & Accommodations**: Featured luxury hotels with ratings and pricing
- 🏥 **Healthcare**: Top hospitals and medical facilities (PGIMER, Fortis, Max Hospital)
- 🎓 **Educational Institutions**: Universities and colleges (PU, PEC, Chitkara)
- 🏛️ **Popular Places**: Tourist attractions (Rock Garden, Sukhna Lake, Rose Garden)
- 🛍️ **Markets & Shopping**: Local markets and shopping destinations
- 🍽️ **Restaurants**: Dining options and food services

### User Experience
- **Responsive Design**: Mobile-first approach with adaptive layouts
- **Dark/Light Mode**: Theme switching for better user experience
- **Interactive Navigation**: Smooth side menu and quick access buttons
- **Team Integration**: Meet the SJR team behind the project

## 🛠️ Technical Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with Google Fonts (Poppins)
- **Icons & Images**: Custom icon set for consistent UI
- **Responsive**: Mobile-optimized design

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd city-sathi
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your preferred browser
   # Or use a local server
   python -m http.server 8000
   # Navigate to http://localhost:8000
   ```

3. **Required Assets**
   Ensure all image assets are properly placed in the project directory:
   - Logo files (`logo.png`, `logo_main.png`)
   - UI icons (menu, mode, search, etc.)
   - Location images for popular places
   - Hotel images
   - Team member photos
